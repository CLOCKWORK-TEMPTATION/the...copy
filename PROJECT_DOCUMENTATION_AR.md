# التوثيق الشامل لمنصة النسخة (The Copy)

## نظرة عامة على المنصة

**النسخة (The Copy)** هي منصة متكاملة للتحليل الدرامي وتطوير السيناريوهات مصممة خصيصاً للمبدعين العرب في مجال الدراما. تجمع المنصة بين قوة الذكاء الاصطناعي وأدوات الإنتاج الاحترافية لتبسيط العملية الإبداعية من كتابة السيناريو حتى الإنتاج النهائي.

### الهدف الرئيسي للمنصة

تهدف منصة "النسخة" إلى تمكين الكتّاب والمخرجين والممثلين والفرق الإنتاجية من خلال:

1. **التحليل الدرامي المدعوم بالذكاء الاصطناعي**: تحليل شامل للشخصيات، البنية الدرامية، الحوار، والسياق الثقافي باستخدام نظام متعدد الوكلاء (Seven-Agent System)
2. **أدوات احترافية للإنتاج**: من الكتابة الإبداعية إلى التصوير التجريبي (Self-Tape) وإدارة المشاريع الإنتاجية
3. **دعم كامل للغة العربية**: واجهة RTL مع تحسينات خاصة للمحتوى العربي
4. **التعاون الفوري**: نظام تحديثات فورية عبر Socket.io لفرق العمل
5. **تحسين الأداء**: استخدام تقنيات متقدمة مثل Web Workers، Redis Caching، وتقسيم الكود

### البنية التقنية

#### Frontend
- **إطار العمل**: Next.js 16 مع React 19
- **لغة البرمجة**: TypeScript 5.7+
- **التصميم**: Tailwind CSS v4 مع نظام ألوان OKLCH
- **المكونات**: Radix UI (Shadcn/ui)
- **الرسوم المتحركة**: GSAP + Framer Motion + Three.js
- **إدارة الحالة**: Zustand (حالة العميل) + React Query (حالة الخادم)
- **المنفذ**: 5000

#### Backend
- **إطار العمل**: Express.js 5 مع TypeScript
- **قاعدة البيانات**: Neon PostgreSQL (Serverless) مع Drizzle ORM
- **التخزين المؤقت**: Redis
- **الذكاء الاصطناعي**: Google Gemini AI + Groq SDK + Genkit
- **المهام الخلفية**: BullMQ
- **الاتصال الفوري**: Socket.io
- **المنفذ**: 3000

#### المراقبة والأمان
- **تتبع الأخطاء**: Sentry
- **التتبع الموزع**: OpenTelemetry
- **السجلات**: Winston
- **المقاييس**: Prometheus + Grafana

## نظام الوكلاء السبعة للتحليل الدرامي

تعتمد المنصة على نظام متطور من سبعة وكلاء متخصصين في تحليل السيناريوهات الدرامية:

### 1. Character Deep Analyzer (محلل الشخصيات المتعمق)
- تحليل شامل لتطور الشخصيات وأقواسها الدرامية
- رسم العلاقات بين الشخصيات
- تقييم الدوافع والصراعات الداخلية

### 2. Dialogue Advanced Analyzer (محلل الحوار المتقدم)
- تحليل أنماط الحوار والأصالة
- تقييم مدى ملاءمة الحوار لكل شخصية
- التحقق من الطبيعية والواقعية

### 3. Cultural Historical Analyzer (محلل السياق الثقافي والتاريخي)
- فحص الدقة الثقافية والتاريخية
- التحقق من مناسبة السياق للحقبة الزمنية
- تقييم الحساسية الثقافية

### 4. Target Audience Analyzer (محلل الجمهور المستهدف)
- تحديد الفئة الديموغرافية المستهدفة
- تقييم مدى ملاءمة المحتوى للجمهور
- اقتراحات للوصول الأمثل للجمهور

### 5. Themes Messages Analyzer (محلل الثيمات والرسائل)
- استخراج الثيمات الرئيسية والفرعية
- تحليل الرسائل الضمنية والصريحة
- تقييم عمق وتماسك الثيمات

### 6. Visual Cinematic Analyzer (محلل البصريات السينمائية)
- اقتراحات للإخراج البصري
- تحليل إمكانيات السرد البصري
- توصيات لزوايا الكاميرا والإضاءة

### 7. Producibility Analyzer (محلل قابلية الإنتاج)
- تقدير الميزانية المطلوبة
- تحليل التحديات الإنتاجية
- اقتراحات لتقليل التكاليف

- **نظام التعاون**: مشاركة المشروع مع الفريق

## متطلبات التشغيل

### البيئة التطويرية
- Node.js 20.x+
- pnpm 10.20.0+
- PostgreSQL (يُنصح باستخدام Neon)
- Redis 7.x+

### متغيرات البيئة المطلوبة
```env
# قاعدة البيانات
DATABASE_URL=postgresql://...
DIRECT_URL=postgresql://...

# Redis
REDIS_URL=redis://localhost:6379

# الذكاء الاصطناعي
GOOGLE_GENAI_API_KEY=your_key
GROQ_API_KEY=your_key

# المصادقة
JWT_SECRET=your_secret
SESSION_SECRET=your_secret

# المراقبة
SENTRY_DSN=your_dsn
SENTRY_ORG=your_org
SENTRY_PROJECT=your_project

# واجهة أمامية
NEXT_PUBLIC_API_URL=http://localhost:3000
NEXT_PUBLIC_SOCKET_URL=ws://localhost:3000
```

### أوامر البدء السريع
```bash
# تثبيت التبعيات
pnpm install

# بدء جميع الخدمات
pnpm start

# بدء بيئة التطوير
pnpm start:dev

# بدء Redis فقط
pnpm start:redis

# إيقاف جميع الخدمات
pnpm kill:dev
```

### أوامر الواجهة الأمامية
```bash
cd frontend

# التطوير
pnpm dev

# البناء
pnpm build

# الاختبار
pnpm test
pnpm e2e

# الفحص
pnpm typecheck
pnpm lint
```

### أوامر الخادم الخلفي
```bash
# من المجلد الجذر
pnpm --filter @the-copy/backend dev
pnpm --filter @the-copy/backend build
pnpm --filter @the-copy/backend test

# قاعدة البيانات
pnpm --filter @the-copy/backend db:generate
pnpm --filter @the-copy/backend db:push
pnpm --filter @the-copy/backend db:studio
```

## الخاتمة

### الرؤية المستقبلية

- تطوير تطبيق محمول
- دعم المزيد من اللغات
- تكامل مع منصات الإنتاج الرئيسية

### الجمهور المستهدف الإجمالي
- الكتّاب والسيناريست
- المخرجون ومديرو الإنتاج
