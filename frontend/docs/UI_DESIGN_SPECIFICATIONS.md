# مواصفات تصميم واجهة المستخدم - منصة النسخة

## نظام التصميم العام (Design System)

### الألوان الأساسية

```css
--background: #000000 (أسود)
--foreground: #FFFFFF (أبيض)
--accent: #FFD700 (ذهبي)
--primary: hsl(var(--primary))
--muted: rgba(255,255,255,0.05)
--border: rgba(255,255,255,0.1)
```

### الخطوط

- **العناوين**: خط عربي عريض (Cairo, Tajawal)
- **النصوص**: Inter, system-ui
- **الاتجاه**: RTL

### المبادئ العامة

1. خلفية سوداء موحدة
2. تدرجات لونية خفيفة (من أزرق لبنفسجي)
3. حواف مستديرة (rounded-lg)
4. ظلال خفيفة
5. أنيميشن سلسة عند التفاعل
6. Cards مع حدود شفافة

## ملاحظات التنفيذ

### المكونات المشتركة

1. `Card`, `CardHeader`, `CardContent` من shadcn/ui
2. `Button` بأحجام مختلفة
3. `Badge` للحالات والتصنيفات
4. `Tabs` للتنقل
5. `ScrollArea` للمحتوى الطويل
6. `Textarea` للإدخال
7. `Tooltip` للتلميحات

### الأيقونات

استخدام مكتبة `lucide-react` لجميع الأيقونات

### الاستجابة

- Mobile: عمود واحد
- Tablet: عمودين
- Desktop: 3+ أعمدة

### الأنيميشن

- `transition-all duration-300`
- `hover:scale-105` للبطاقات
- `animate-pulse` للعمليات الجارية
